## Description

Details the purpose, operation, and selection criteria of control transformers, vital for supplying safe, isolated control voltage to industrial control circuits.

## Core Concepts

### What is it?

A transformer designed to step down high distribution voltage to lower, safer voltages for operating control components.

### How it works

Uses magnetic induction to convert high input voltage (e.g., 480VAC) into lower output voltages (e.g., 120VAC or 24VAC).

### When to Use

Use in control systems requiring isolated, lower-voltage power sources.

## Specifications & Parameters

- Typical Ratings: 50VA to 1000VA
    
- Input Voltage: 240VAC, 480VAC
    
- Output Voltage: 24VAC, 120VAC
