Description

Pulling the PLC, HMI, and drives together into a single working system is its own skill. Each alone is simple; the integration is where pieces disagree on protocol, timing, addresses, and fault handling.

Integration Checklist

  1. IP plan published and matched by all devices. See IP Addressing.
  2. PLC communication modules configured for the drive and HMI protocols.
  3. Drive added to PLC controller organizer with correct EDS / GSD / ESI file.
  4. Connection parameters (RPI, slot, connection timeout) tuned.
  5. HMI connected, tags imported, faceplates instantiated.
  6. Alarms propagated from drive faults to PLC and onward to HMI.
  7. Commands and status mapped both ways for drive control.
  8. Speed reference path validated (local vs. remote).
  9. Safety interlocks verified — drive Safe-Torque-Off, emergency stop.
  10. Recovery from power cycle and network loss tested.

Common Pitfalls

  • EDS file revision doesn't match drive firmware — configuration errors appear subtle.
  • Command source parameter left in keypad mode after bench testing.
  • HMI button writes directly to the drive tag, bypassing PLC sequence logic.
  • Drive's network-loss reaction set to 'maintain last speed' for a hoist.
  • Drive parameters not saved to flash after commissioning — first power cycle clears them.
  • Time sync missing — alarms across PLC / drive / HMI don't line up.

Recommended Architecture

HMI ---(tags)--- PLC ---(implicit I/O)--- Drive
                 |
                 +-- state machine / interlocks / sequence
                 |
                 +-- alarm aggregation

Operator buttons go to PLC tags, NEVER directly to drive control bits.
PLC is the single source of truth for drive commands.

Field Checklist

  • ☐ Parameter file archived for each drive.
  • ☐ PLC program archived; HMI project archived.
  • ☐ Alarm propagation verified by deliberate fault injection.
  • ☐ Manual override (local/remote) tested.
  • ☐ System runs through a complete cycle with no operator intervention.
  • ☐ Commissioning report signed — see Commissioning Reports.
